# 🏥 MediVault-V1  

Tech-forward medical record management platform built using Angular, Node.js, Express.js, and MongoDB Atlas.

---

## 🚀 Welcome to MediVault  

### 📌 Overview  
MediVault is a modern web application designed to simplify how individuals store and manage their medical records.

Instead of relying on physical documents, MediVault provides a secure digital platform where users can upload, access, and organize their health data anytime, anywhere.

In addition to personal records, users can also manage and store medical data for multiple family members, making it easier to track and access healthcare information for the entire household from a single account.

It focuses on security, accessibility, and ease of use, making healthcare record management efficient and reliable.

---

## 🌟 Key Features  

- 🔐 User Authentication – Secure login and registration system  
- 📁 Medical Record Management – Store all your health documents in one place  
- 📄 File Upload System – Upload prescriptions, reports, and medical files  
- 👤 User Dashboard – Personalized space to manage and view records  
- 🔍 Quick Search – Easily find specific medical documents
