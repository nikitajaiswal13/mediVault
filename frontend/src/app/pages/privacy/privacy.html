1. Introduction

MediVault is a secure medical record management platform designed to help users store and manage healthcare documents digitally. This Privacy Policy explains how we collect, use, and protect your information.

2. Information We Collect

  • User account information (name, email).
  • Patient details entered by users.
  • Medical records and uploaded documents.
  • Authentication data for secure login sessions.

3. How We Use Information

  • Provide secure dashboard access.
  • Manage and organize patient records.
  • Maintain authentication and system security.
  • Improve application performance.

4. Data Security

MediVault uses JWT-based authentication, protected API routes, and user-specific data isolation to ensure records remain accessible only to authorized users.

5. Data Sharing

MediVault does not sell, rent, or share user data with third parties. All information remains private and accessible only to the authenticated account holder.

6. File Storage

Uploaded medical documents are securely stored and access is restricted using authentication and ownership validation.

7. User Rights

  • Users can delete patients and records anytime.
  • Users may stop using the platform whenever they choose.

8. Disclaimer

MediVault is an educational demonstration project and is not a certified healthcare system. It does not provide medical advice or professional services.
